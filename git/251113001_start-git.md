# Shell, Vim commands

## Shell Command

- ls: list
- cd: change directory
- mkdir: make directory
- pwd: print working directory

- touch newfile.md: make newfile
- mv newfile.md dev/: move newfile to dev/
- mv newfile.md Newfile.md: change filename from newfile to Newfile
- cp Newfile.md Newfile_copy.md: copy Newfile.md as Newfile_copy.md
- rm Newfile.md: remove file
- rm -rf dev: remove folder
- cat Newfile.md: display file contents

## Vim Command

### normal mode

- i: insert mode
- ESC: back to normal mode

### Cmd-line mode

- :q! -> override and quit
- :wq -> write and quit
