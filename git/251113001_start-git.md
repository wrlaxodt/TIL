# Shell, Vim commands

## Shell Command

- ls: list
- cd: change directory
- mkdir: make directory
- pwd: print working directory

- touch newfile.md: make newfile
- mv newfile.md dev/: move newfile to dev/
- mv newfile.md Newfile.md: change filename from newfile to Newfile
- cp Newfile.md Newfile_copy.md: copy Newfile.md as Newfile_copy.md
- rm Newfile.md: remove file
- rm -rf dev: remove folder recursively and forcefully
- cat Newfile.md: print file contents

## Vim Command

### normal mode

- i: insert mode
- ESC: back to normal mode

### Cmd-line mode

- :q! -> quit without saving
- :wq -> write and quit

# git

## Set configuration

- $ git config --global user.name "wrlaxodt"
- $ git config --global user.email "wrlaxodt@gmail.com"
- $ git config --global core.editor "vim"
- $ git config --global core.pager "cat"

## How to start

- $ git clone {repo-addr} -> load repository
- $ cd {repo-addr} -> change directory to loaded repository

- $ vi README.md
- $ git status
- $ git add README.md -> from Working Directory to Staging Area
- $ git commit -> from Staging Area to Local Repository
- $ git push origin main -> from Local Repository to Remote Repository

## .gitignore

List files and folders to exclude from Git.
Use [gitignore.io](https://www.toptal.com/developers/gitignore/)

## pre-commit

Runs automated checks before each commit.

- $ git add README.md
- $ pre-commit run -> detects issues and fixes them automatically
- $ git add README.md -> re-add the fixed file
- $ git commit
