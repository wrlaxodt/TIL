# Branch

A separate workspace to make independent code changes.

## Branch list

- (local) $ git branch
- (remote) $ git branch -r
- (all) $ git branch -a

## Create new branch

- $ git branch {branch-name}

## Switch to branch

- $ git switch {branch-name}

## Merge branch

- $ git merge {branch-name}

## Delete branch

- $ git branch -D {branch-name}
